# Publications

My research contributions focus on human-robot collaboration, augmented reality-based manufacturing systems, intelligent manufacturing, and human-centered Industry 5.0 technologies.

## Selected Journal Publications

### Multi-modal empowerment for human-robot collaboration team: A hidden semi-Markov model and augmented reality-based collaborative assembly system

This research develops an adaptive human-robot collaboration framework by integrating multimodal human state recognition, hidden semi-Markov modeling, and augmented reality technologies to improve collaborative assembly performance.

**Research areas:** Human-Robot Collaboration, Augmented Reality, Human Factors Engineering, Intelligent Manufacturing

---

### Intelligent task allocation and human-robot collaboration using fuzzy logic approaches

This study investigates adaptive task allocation strategies between humans and robots using intelligent decision-making methods for collaborative manufacturing environments.

**Research areas:** Robotics, Artificial Intelligence, Task Allocation, Industry 5.0
